¿Qué es la autenticación en dos pasos?

La autenticación en dos pasos añade una capa adicional de seguridad a tu cuenta. Además de tu contraseña, necesitarás un código generado por una aplicación de autenticación.

Aplicaciones recomendadas

phone_iphone Google Authenticator
android Authy
security Microsoft Authenticator

Cómo configurar

  1. Descarga una aplicación de autenticación
  2. Escanea el código QR en tu perfil
  3. Guarda los códigos de respaldo
  4. Ingresa el código de 6 dígitos cuando se solicite
